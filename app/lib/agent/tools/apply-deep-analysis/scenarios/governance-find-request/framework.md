# General Codebook

## Scope

- Code only government speakers (ministers, officials). Treat
  journalist questions as context unless the answer can't be
  understood without them.
- When a passage is clearly off-topic from COVID governance,
  leave it uncoded.

## Threshold

- Prefer the narrower, defensible reading. When a passage doesn't
  clearly meet a code's inclusion criteria, leave it uncoded.
