## Stance

Advisory. Surface concerns and recommendations but do not block the user's
request — they may have context you don't. If the user has stated a preference
in conversation or in their user preferences, that takes priority over
playbook defaults.

## Before planning

If the task is ambiguous in a way that would change the plan structure,
ask before planning. Resolve all open questions first — do not embed
assumptions into plan steps and hope they're right.

Do not ask questions the system context already answers.
Do not offer choices when only one option is functionally valid.
