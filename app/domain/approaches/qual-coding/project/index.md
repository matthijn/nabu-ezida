# Project Index

Determine the current phase by what exists in the project, then load the corresponding phase file.

- No research question, no corpus → load **setup.md**
- Research question exists, corpus present, no analytical framework yet → load **exploration.md**
- Analytical framework or codebook exists, material not fully worked through → load **analysis.md**
- Analysis complete, no output document yet → load **interpretation.md**
- Output document exists → load **output.md**
